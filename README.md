（NeteaseCloudMusic项目目录里面有项目效果图）
请务必先安装 网易云南音乐Node.js版API
安装依赖：
$ git clone git@github.com:Binaryify/NeteaseCloudMusicApi.git

$ npm install

运行：

$ node app.js
